# Requisitos del Perfil - Backend Developer

## Requisitos Obligatorios

| Competencia | Nivel Requerido | Descripcion |
|-------------|----------------|-------------|
| Node.js / TypeScript | Senior (3+ anos) | Dominio de async/await, streams, event loop, manejo de errores |
| NestJS | Intermedio-Senior (2+ anos) | Modulos, providers, guards, interceptors, pipes, decorators custom |
| Express.js | Intermedio (2+ anos) | Middleware, routing, manejo de errores, integracion con ORMs |
| PostgreSQL | Intermedio-Senior (2+ anos) | Queries complejas, indices, schemas, transacciones, optimizacion |
| Prisma ORM | Intermedio (1+ ano) | Schema design, migraciones, relaciones, raw queries, multi-schema |
| REST API Design | Senior (3+ anos) | Versionado, paginacion, codigos HTTP, autenticacion, documentacion |
| Git | Intermedio (2+ anos) | Branching strategies, rebases, cherry-pick, resolucion de conflictos |
| Docker | Intermedio (1+ ano) | Dockerfiles multi-stage, docker-compose, networking entre contenedores |

## Requisitos Deseables

| Competencia | Nivel | Descripcion |
|-------------|-------|-------------|
| Kubernetes | Basico-Intermedio | Pods, deployments, services, namespaces, ConfigMaps, Secrets |
| Arquitectura de microservicios | Intermedio | Comunicacion entre servicios, patrones (saga, circuit breaker, API gateway) |
| Multi-tenancy | Basico | Schema-based isolation, tenant resolution, connection pooling |
| CI/CD | Basico | Jenkins pipelines, GitHub Actions, despliegue automatizado |
| Terraform / IaC | Basico | Lectura y modificacion de modulos existentes |
| Message queues | Basico | RabbitMQ, Redis pub/sub, o similares |
| Pasarelas de pago | Deseable | Integracion con proveedores de pago, manejo de webhooks, reconciliacion |

## Competencias Blandas

- Capacidad de trabajar con codigo legacy y codigo moderno en el mismo proyecto.
- Autonomia para investigar y resolver problemas sin documentacion exhaustiva.
- Comunicacion clara para reportar avances y bloqueantes.
- Disposicion para aprender tecnologias nuevas rapidamente.

## Nivel General Esperado

El candidato debe tener un minimo de **3 anos de experiencia** en desarrollo backend con Node.js. Debe sentirse comodo tanto creando servicios desde cero como manteniendo y mejorando servicios existentes. Se espera que pueda tomar decisiones de arquitectura fundamentadas y comunicarlas al equipo.
